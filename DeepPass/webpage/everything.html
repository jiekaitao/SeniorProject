2. Track A: DeepPass layer duplication

Build on David Ng's RYS (Repeat Your Self): duplicate specific transformer blocks at inference, blend the seam with per-layer alpha, and score climbs +7.31 over Ng's published result on Qwen2-72B. Zero training, zero new weights.

Candidate blocks are chosen by spectral screening. A block [i,j) is a good duplication target when passing hidden states through its own function again is close to a no-op after subtracting the one-step update direction:

$$\rho(i,j) = \|F_{i:j}(F_{i:j}(h)) - F_{i:j}(h)\| \,/\, \|F_{i:j}(h) - h\|$$

We combine this with BLOOD-impact (a gradient-based sensitivity) to rank blocks. SBUID_0 achieves Spearman r=0.515, p=0.008 on Qwen2-72B:

$$\text{SBUID}_0 = \text{BLOOD-impact} - 6000 \cdot \rho$$

The seam between a block's original output and its re-application gets a per-layer alpha blend:

$$h_{\text{seam}} = h_1 + \sum_l \alpha_l \,(h_2^{(l)} - h_1^{(l)})$$

FFN re-retrieval corruption Duplication helps reasoning (IFEval +2.3%, MuSR +1.3%) but hurts knowledge (MATH −6.4%). Sublayer analysis isolates the culprit: attention benefits from repetition (re-weighting an already-attended context), but FFN hurts because it runs key-value retrieval on a perturbed query, pulling in the wrong facts. Attention-only duplication should fix this; FFN-only duplication makes it worse.

3. Track B: recurrent deliberation controller

A small (~140M) learned controller that writes iterative thought tokens into the residual stream of a frozen Llama 3.1 8B. Read from a fixed set of tapped layers, update state, splice thoughts back in, iterate K rounds, decode from the unchanged LM head.

[prompt] layers 0..11 [prompt|thoughts|answer] layer 12..31 ↓ ↓ ↓ ↓ embed tokens → frozen LM layers → splice thoughts → frozen LM layers → ABCD logits ↑ │ z ∈ R^(8×512) —write—┘ │ ↑ │ RMSNorm(z + σ·Δ) ←— read features ←——————————————————┘ (tapped layers 8,16,24 + slot hidden states) [iterate K rounds]

Key equations

Thought write (vocab-distilled + low-rank residual):

$$\text{thought}(z) = \sum_{k=1}^{64} \text{softmax}(\text{logits}_k(z)) \cdot E[\text{topk}_k(z)] + 0.12 \cdot z\, W_\downarrow W_\uparrow$$

State update:

$$z \leftarrow \text{RMSNorm}\big(z + \sigma \cdot \text{read\_proj}(\text{features})\big)$$

Training loss (final CE + per-round value head + anti-degeneracy margin):

$$\mathcal{L} = \text{CE}_{\text{final}} + 0.5 \cdot \frac{1}{R}\sum_r \text{BCE}(v_r, \text{correct}_r) + 0.1 \cdot \max(0, \text{CE}_{\text{final}} - \text{CE}_{\text{first}} + 0.1)$$
